# FoodTrends: Understanding Customer Preferences in Facebook
# Project Overview

This project focuses on analyzing customer engagement patterns on Facebook posts related to food categories. The main objective is to understand how different types of content, posting strategies, and food categories influence audience interaction.

Using data science techniques, the project explores engagement metrics such as likes, comments, shares, and total engagement to derive meaningful insights about user preferences.

# Objectives

- To analyze user engagement across different food categories

- To identify patterns in audience interaction based on post type and timing

- To understand which factors contribute to higher engagement

- To build visual dashboards for better interpretation of insights

# Dataset Description

The dataset contains information about Facebook food-related posts, including:

- Post ID

- Food Category (Pizza, Burger, Sushi, Salad)

- Brand Name

- Likes Count

- Comments Count

- Shares Count

- Total Engagement

- Time of Posting

- Day of Posting

# Data Preprocessing

Before analysis, the dataset was cleaned and prepared by:

- Handling missing values

- Removing duplicate records

- Converting data types where necessary

- Creating a new feature: Total Engagement (sum of likes, comments, and shares)

This step ensured that the dataset was accurate and ready for analysis.

# Exploratory Data Analysis (EDA)

EDA was performed to understand the data distribution and relationships between variables.

Key techniques used:

- Histogram to analyze engagement distribution

- Boxplot to detect outliers

- Bar charts for comparing engagement across categories

- Scatter plots to study relationships between likes, comments, shares, and total engagement

- Correlation heatmap to identify strong relationships

# Data Visualization

Data visualization was used to present insights clearly and effectively.

Visualizations include:

- Engagement by food category

- Engagement by brand

- Engagement by time of day

- Engagement by post type (image vs video)

These visuals help in identifying trends and comparing performance across different dimensions.

# Dashboard Development (Power BI)

An interactive dashboard was created using Microsoft Power BI to present insights in a user-friendly manner.

Dashboard features:

- Key performance indicators (KPIs) such as total engagement, likes, comments, and shares

- Filters for food category and day of the week

- Visual comparisons across categories, brands, and time periods

- Interactive charts for dynamic exploration

# Key Insights

Certain food categories generate higher engagement than others

- Video posts slightly outperform image posts in engagement

- Posting time significantly affects audience interaction

- Likes have a strong correlation with total engagement

- Brand popularity influences engagement levels

# Conclusion

The analysis highlights how customer engagement varies across different factors such as content type, posting time, and food category. The use of data visualization and dashboards makes it easier to interpret patterns and derive actionable insights.
